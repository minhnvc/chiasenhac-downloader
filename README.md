## Installation
`$ npm i -g chiasenhac-downloader`

## Download Album
`$ chiasenhac album {email} {password} {album-link}`

### Example
`$ chiasenhac album abc@gmail.com abc123 https://chiasenhac.vn/nghe-album/different-world-xss7r6rsqtaka4.html`
